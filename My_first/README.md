# 🎬 Проект "Кінотеатр"

Django проект для управління кінотеатром з фільмами, сеансами та відгуками.

## 📋 Функціонал

### ✅ Моделі

1. **Movie (Фільм)**
   - Назва, опис, рік випуску
   - Тривалість, жанр
   - Постер (ImageField)
   - Автоматичний підрахунок середньої оцінки

2. **Session (Сеанс)**
   - Зв'язок many-to-one з Movie
   - Дата та час сеансу
   - Номер зали (1-20)

3. **Review (Відгук)**
   - Зв'язок many-to-one з Movie
   - Ім'я користувача
   - Текст відгуку
   - Оцінка (1-10)
   - Дата створення

### ✅ Функції

- **CRUD для фільмів**: створення, перегляд, редагування, видалення
- **CRUD для сеансів**: створення та видалення сеансів
- **Система відгуків**: додавання та видалення відгуків прямо на сторінці фільму
- **Перегляд сеансів**: окрема сторінка з усіма найближчими сеансами
- **Адаптивний дизайн**: Bootstrap 5
- **Постери фільмів**: завантаження та відображення

## 🚀 Встановлення та запуск

### 1. Встановлення залежностей

```bash
cd My_first
pip install Pillow
```

### 2. Створення та застосування міграцій

```bash
python manage.py makemigrations
python manage.py migrate
```

### 3. Створення суперюзера

```bash
python manage.py createsuperuser
```

**Або використайте готовий:**
- Username: `admin`
- Password: `admin123`
- Email: `admin@cinema.ua`

### 4. Запуск сервера

```bash
python manage.py runserver
```

Відкрийте браузер: http://127.0.0.1:8000/

## 🎯 Основні URL

| URL | Опис |
|-----|------|
| `/` | Список всіх фільмів |
| `/movie/<id>/` | Деталі фільму з сеансами та відгуками |
| `/movie/create/` | Додати новий фільм |
| `/movie/<id>/edit/` | Редагувати фільм |
| `/movie/<id>/delete/` | Видалити фільм |
| `/sessions/` | Всі найближчі сеанси |
| `/session/create/` | Додати сеанс |
| `/movie/<id>/session/create/` | Додати сеанс для конкретного фільму |
| `/session/<id>/delete/` | Видалити сеанс |
| `/review/<id>/delete/` | Видалити відгук |
| `/admin/` | Адміністративна панель |

## 💡 Особливості реалізації

### 1. Зв'язки між моделями

- **One-to-Many (Movie → Session)**: один фільм може мати кілька сеансів
- **One-to-Many (Movie → Review)**: один фільм може мати кілька відгуків

### 2. Відображення відгуків

Відгуки додаються та відображаються прямо на сторінці фільму:
- Форма для додавання відгуку
- Список усіх відгуків з оцінками
- Середня оцінка фільму

### 3. Сеанси

- Автоматична фільтрація: показуються тільки майбутні сеанси
- Сортування за датою
- Можливість створити сеанс прямо зі сторінки фільму

### 4. Постери

- Завантаження через форму
- Відображення через `{% static %}` та `{{ movie.poster.url }}`
- Плейсхолдер з іконкою, якщо постер не завантажено

### 5. Дизайн

- Адаптивний Bootstrap 5
- Font Awesome іконки
- Кастомні CSS стилі
- Анімації та ефекти hover
